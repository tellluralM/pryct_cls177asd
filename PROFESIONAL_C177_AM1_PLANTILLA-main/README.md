# PROFESIONAL_C177_PLANTILLA (MAESTRA)
Crea un entorno virtual usando los siguientes comandos para ejecutar este proyecto:
#### Windows:
python3.8 -m venv venv

venv\Scripts\activate.bat

pip install flask

python app.py

#### MacOS/Linux:

python3.8 -m venv venv

venv/Scripts/activate

pip install flask

python app.py
